# Directive: Project Overview - R.F. Resina Forlivese (Cinematic Experience)

## Objective
Create an immersive, high-impact cinematic web experience for R.F. Resina Forlivese S.r.l. (since 1973). The goal is to convey industrial power, Italian elegance, and artisanal precision.

## Design Philosophy
- **Impact**: Cinematic (Apple/Porsche style), 100vh storytelling.
- **Motion**: Scroll-driven animations, 3D tilt, reveal-on-scroll, smooth page transitions.
- **UX**: Custom cursor follower, magnetic buttons, parallax effects.

## Brand Identity
- **Colors**:
  - Primary: Dark Navy (#1a2744)
  - Secondary: Anthracite (#2d2d2d)
  - Accents: Steel Blue (#4a6fa5), Gold/Champagne (#c9a96e)
  - Backgrounds: Deep Black (#0a0a0a) & Pure White (#ffffff) alternated.
  - Sustainability: Green (#4CAF50).
- **Typography**:
  - Headings: Clash Display / Satoshi / Bebas Neue (Bold/Uppercase).
  - Body: Inter / DM Sans.
  - Numbers: Monospace or Display (Counter anim).

## Detailed Section Structure
1.  **HERO (100vh)**: Video loop, reveal animation for "PLASMIAMO LA MATERIA DAL 1973", magnetic CTA.
2.  **NUMBERS**: Sticky scroll, count-up counters (50+ years, 1973, 100% Made in Italy).
3.  **CHI SIAMO**: Split screen parallax, interactive horizontal timeline.
4.  **SETTORI**: Interactive grid with hover 3D tilt cards.
5.  **PRODOTTI**: Horizontal scroll gallery triggered by vertical scroll.
6.  **SERVIZI**: Scroll reveal with animated SVG line icons.
7.  **PRODUZIONE**: Masonry/Lightbox gallery of the factory.
8.  **QUALITÀ**: Manifesto-style manifesto with ISO 9001 certifications.
9.  **SOSTENIBILITÀ**: Accent green section with "Pensa all'ambiente" message.
10. **CONTATTI**: Split screen with dark-mode Map and animated Form.

## Technical Stack
- **Frontend**: Next.js 15, Tailwind CSS v4, Framer Motion, Lenis (Smooth Scroll).
- **Backend**: FastAPI.
- **Icons**: Lucide React / SVG animations.
